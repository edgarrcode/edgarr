Web Design For UTEP Enterprise Computing

When working at the UTEP Enterprise Computing team I created an HTML/CSS/JS framework to be used to build custom web applications.This framework improved the process by reducing the time of building the front end of the applications.

This framework has been used to build a variety of web applications including sign-on pages, data logging applications, and traffic ticket payment method.

UTEP Enrollment Services

This project consisted in creating a DNN theme that would be used for different departments of The University of Texas at El Paso. The Enrollment Services department provided the design in Illustrator files and I created the HTML/CSS templates and the DNN theme.
